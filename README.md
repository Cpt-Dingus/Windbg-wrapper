# Windbg-wrapper
Wraps CDB (Command line version of WinDBG) in a tkinter interface that allows tabbed debugging. Support .zip, .dmp and folders contaning .dmp files.
#### **Depends on the -SDK- version of WinDBG/debugging tools**
